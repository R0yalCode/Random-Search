Random Search

¿Qué es Random Search?
Random Search es una forma de explorar y probar distintas opciones en problemas complejos, eligiendo soluciones al azar dentro de todo el espacio posible.
Es como lanzar dardos hacia una diana gigante, pero sólo unos pocos caen (probar todo es imposible).
Analogía Visual: El Garage de Equipos
El “garage” es el universo de combinaciones de modelos posibles.
Random Search escoge algunos autos al azar y busca el ganador. Verlo ayuda a entender la diferencia frente a buscar “todos los autos”.
Funcionamiento Secuencial de Random Search
Cada etapa: definir espacio, seleccionar aleatorio, evaluar, elegir... El ciclo fundamental de Random Search.
Un árbol de decisión (ejemplo)
Cada configuración probada por Random Search genera “árboles binarios” (mecanismos reales de diagnóstico automático).

Random Óptimo Grid
Coste computacional y eficiencia (Big O)
Random Search: O(k·C)   Grid Search: O(g·C)
“k” = intentos random, “g” = toda la grilla, “C” = coste de entrenar/evaluar un modelo.
Cuanto más grande el espacio, más ahorro de Random Search.
